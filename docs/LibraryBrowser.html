Library Browser Online

Library Browser is an application from Open Source Physics (OSP) that enables you to browse and open collections of digital library resources such as EJS models and Tracker experiments. To open a collection choose from the Collections menu.

You can also build your own collection using the Java version of Library Browser included with desktop Tracker. Collections are saved as xml files with references to the actual resources. For more information see Library Browser Help.

You can reposition Library Browser Online by dragging its title bar and resize it by dragging the lower right hand corner.

Library Browser was written by Douglas Brown at Cabrillo College using portions of the Open Source Physics Java code library developed by Wolfgang Christian at Davidson College. Library Browser was later converted from Java to JavaScript by Douglas Brown, Wolfgang Christian and Robert Hanson using the SwingJS system developed by Hanson and his students at St. Olaf College. This web page runs the JavaScript conversion.
